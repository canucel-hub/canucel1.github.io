  • Scratch XOX Project : How it works: This is a tic-tac-toe (XOX) game built in Scratch where two players take turns placing “X” and “O” on a 3×3 board. 1 click for X, 2 clicks for O What I have done: I created the board using Scratch buttons, added costumes and made change the costume everytime someone clicks. I also made the game restartable by clicking space or green flag so it can be played multiple times. Was it hard? Overall Scratch made it easier since it uses block coding.
